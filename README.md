## 📖 Présentation

**Kadea Chat** est une application web développée dans le cadre du **Capstone 1 de Kadea Academy**. Elle reproduit les principales fonctionnalités de **WhatsApp Web** en consommant une API REST.

L'objectif du projet est de mettre en pratique les compétences en développement Front-End, la manipulation des API REST, la gestion des utilisateurs et des conversations, ainsi que l'utilisation de Git et GitHub dans un environnement professionnel.

---

## ✨ Fonctionnalités principales

- 🔐 Authentification (inscription, connexion, déconnexion)
- 👤 Gestion du profil utilisateur
- 💬 Affichage des conversations
- 📨 Envoi et réception de messages
- ✏️ Modification des messages
- 🗑️ Suppression des messages
- 😀 Sélecteur d'emojis
- 📎 Envoi de fichiers
- 📱 Interface responsive inspirée de WhatsApp Web
- 🕒 Affichage intelligent des dates et heures
- 🔄 Actualisation automatique des conversations

# ⚙️ Installation

## Prérequis

Avant de commencer, assurez-vous d'avoir installé :

- Git
- Node.js (version récente)
- npm
- Visual Studio Code (recommandé)

---

## 1. Cloner le dépôt

```bash
git clone https://github.com/kadea-academy-learners/capstone-1-kadea-chat-clone-whatsapp-web-dim-mbolokala.git
```

---

## 2. Accéder au projet

```bash
cd capstone-1-kadea-chat-clone-whatsapp-web-dim-mbolokala
```

---

## 3. Installer les dépendances

Si le projet utilise npm :

```bash
npm install
```

---

## 4. Lancer le projet

Avec Live Server :

- Ouvrir le dossier dans Visual Studio Code
- Installer l'extension **Live Server**
- Faire un clic droit sur `index.html`
- Sélectionner **Open with Live Server**
